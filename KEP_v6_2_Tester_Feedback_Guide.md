# KEP v6.2 — Tester Feedback Pack Guide

## Purpose

KEP now needs feedback from trusted testers before wider launch.

This pack adds:

- `tester-instructions.html`
- `tester-feedback.html`
- `report-question.html`
- `admin-feedback-db.html`
- `feedback.js`
- `scripts/KEP_v6_2_Tester_Feedback_Table.sql`

## Setup

1. Upload v6.2 files to GitHub.
2. Run this SQL in Supabase:

`scripts/KEP_v6_2_Tester_Feedback_Table.sql`

3. Login as a student/tester.
4. Submit one test feedback.
5. Login as admin and open `admin-feedback-db.html`.

## Important

Feedback submission requires login. This avoids random anonymous spam and keeps reports connected to real test users.

## Suggested first testers

Start with 2–3 trusted people only.
