# KEP v6.3 — Simple First Test Plan

Use this for the first trusted testers.

## Test only this

1. Open Start page.
2. Practice 5 questions.
3. Take one mock exam.
4. Check Progress.
5. Send feedback.
6. Report one question if something looks wrong.

## Do not test everything

Do not send testers into admin, developer, setup, AI, or database pages.

The goal is to check whether the student journey feels simple and useful.

## Main questions for testers

- Was it easy to start?
- Did practice work?
- Did mock exam work?
- Did progress update?
- Was anything confusing?
- Would this help a real student?
